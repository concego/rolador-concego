Como usar o rolador

Use o formato XdY, onde X é a quantidade e Y os lados.

  • -L: Exclui o menor resultado.
  • -H: Exclui o maior resultado.
  • + ou -: Soma bônus fixos (Ex: 2d10+5).
  • >= ou <=: Verifica sucesso (Ex: 1d20>=12).
